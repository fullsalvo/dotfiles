Relevant Information
====================

* Window Manager: `i3-gaps`_

* Text Editor: `emacs`_

* File Manager: `ranger`_

* Music Player: `mpd`_ + `ncmpcpp`_

* Browser: Firefox

* IRC: `irssi`_

* Theming: `zenbu`_
